# APK downloads

Place les fichiers APK de Media Vision dans ce dossier.

Les liens actuels dans `applications.html` attendent ces fichiers :

- `occasion.apk`
- `mediavision-news.apk`
- `vision-radio.apk`
- `vision-tv.apk`
- `vision-sport.apk`

Quand un fichier est ajouté, passe `apkAvailable` à `true` pour l'application correspondante dans `applications.html`.
