📲 Comment installer une application téléchargée

  1. Appuyez sur « ⬇ Télécharger l'APK » : le fichier se télécharge directement dans votre dossier « Téléchargements ».
  2. Ouvrez le fichier .apk téléchargé depuis votre gestionnaire de fichiers ou la notification de téléchargement.
  3. Si Android affiche « Installation bloquée », appuyez sur Paramètres puis autorisez « Installer des applications inconnues » pour votre navigateur, puis relancez l'installation.
  4. Une fois installée, ouvrez l'application depuis votre écran d'accueil.
Les boutons de téléchargement APK sont actifs uniquement lorsqu’un fichier officiel est disponible.
